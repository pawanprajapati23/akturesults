## Deployment

### Vercel Deployment

1. Install Vercel CLI: `npm i -g vercel`
2. Navigate to project: `cd akturesults`
3. Deploy: `vercel`
4. Follow prompts
5. Done!

### GitHub Pages Deployment

1. Create GitHub repository
2. Upload all files to repository
3. Go to Settings → Pages
4. Select main branch
5. Save and deploy

### Manual Hosting

Simply upload all files to your web hosting via FTP/cPanel.

## Configuration

Replace domain name in:
- All canonical URLs in HTML files
- sitemap.xml URLs
- Any hardcoded domain references

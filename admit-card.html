How to Download AKTU Admit Card?

  1. AKTU official website par jaye
  2. Student login section me jaye
  3. Roll number aur date of birth enter kare
  4. Admit card download option par click kare
  5. PDF save kar le aur print out le lo

Important Information on Admit Card

  • Student Details: Name, Roll Number, Course
  • Exam Center: Venue address and code
  • Exam Schedule: Date and time for each paper
  • Subject Details: Paper code and subject name
  • Instructions: Exam rules and regulations

What to Do if Admit Card Not Available?

  • College administration se contact kare
  • Exam form properly submit kiya tha ya nahi check kare
  • Fee payment confirm kare
  • AKTU helpline se help le

Important Points

  • Admit card without photo is INVALID
  • Exam center me original photo ID proof le jana compulsory hai
  • Admit card ke sath college ID card bhi le jaye
  • Exam se pehle admit card carefully verify kar lo
  • Koi discrepancy ho to immediately report kare

Documents Required in Exam Hall

  • Printed Admit Card
  • College ID Card
  • Aadhar Card / Any Government ID
  • Photo (if not on admit card)
